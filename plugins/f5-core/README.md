# F5 Core Plugin

Core plugin for the F5 Framework - AI-assisted development with quality gates, TDD support, and workflow optimization.

## Installation

```bash
# Via Claude Code plugin system
/plugin marketplace add Fujigo-Software/f5-framework-claude
/plugin install f5-core
```

## Contents

### Commands (70)
AI-assisted development commands including:
- `/f5-init` - Initialize F5 Framework in workspace
- `/f5-analyze` - Analyze codebase architecture
- `/f5-implement` - Implement features with quality gates
- `/f5-test` - TDD-driven testing workflow
- `/f5-review` - Code review with best practices
- And 60+ more specialized commands

### Agents (25)

### Framework Configs
Quality gates, schemas, templates, and TDD configurations.

## Setup

After installing the plugin, initialize in your workspace:

```bash
/f5-init
```

This copies framework configs to your workspace `.f5/` directory.

## MCP Servers

- Sequential Thinking - Step-by-step reasoning

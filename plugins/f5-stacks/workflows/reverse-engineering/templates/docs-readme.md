## Documentation Standards

### Naming Conventions
- Files: `kebab-case.md`
- Entities: `PascalCase`
- API endpoints: `/api/v1/resource-name`
- Database tables: `snake_case`

### Diagram Standards
- Use Mermaid for all diagrams
- ERD: `erDiagram`
- State machines: `stateDiagram-v2`
- Sequences: `sequenceDiagram`
- Flowcharts: `flowchart TB/LR`

### Code References
Always include source file references:
```markdown
**Source**: `app/Models/Entity.php:method()`
```

## How to Update

1. Run reverse engineering workflow:
   ```bash
   f5 docs analyze
   ```

2. Manual updates:
   - Edit relevant `.md` files
   - Update version in `versions/`

3. Generate version changelog:
   ```bash
   f5 docs version <phase>
   ```
